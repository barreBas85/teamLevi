StappenPlan

1. Go to github 2. Click right above on your avatar
3. Click on "Your repositories"
4. Click right above on "New"
5. Fill your repo name in and set the repo on private or public. If you've done the settings, click on "Create Repository"
6. Go to your repo and click "Clone or Download"
7. Copy the https link and open git bash on your computer
8. Type Git clone (https link) and you will go inside the repo
9. to make a branch you can type git checkout -b "branch name"
10. you can do git add . to add everything
11. you can do git push after that
12. then you need to copy the text that from git push and paste it again in git
13. type git push after this and follow the instructions
14. go to github and there you can see your files uploaded
